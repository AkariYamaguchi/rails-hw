# Tweeto World!
自分のつぶやきを世界に発信。今の気分などを他の人と共有することで新たな発見が見つかるかも知れない。

***デモ***

![デモ](https://tweetoworld.herokuapp.com/)

## 機能

- 他の人の投稿を閲覧出来る
- つぶやける
- 投稿を編集・削除出来る（投稿主だけ）
- 投稿に画像添付して投稿可能
- プロフィールアイコン追加可能
- ユーザ名変更可能


## 使い方

1. 右上にある「ログイン」を押す
2. アカウントがない場合は、「Sign up」を押し、メールアドレスとパスワード入力。
